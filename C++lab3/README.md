## 项目说明

这是一个简化版正则表达式解释器控制台应用，用于完成 “C++ ФИТ 2025 - 任务 3” 中的第三题需求。

### 支持的语法

- **字面量字符**: `a`...`z`, `A`...`Z`, `0`...`9`, 空格 —— 匹配自身
- **点号**: `.` —— 匹配任意单个字符
- **字符组**: `[ ... ]`
  - 例如：`[abc]` 匹配 `a` 或 `b` 或 `c`
  - 范围：`a-d` 表示 `abcd`
- **修饰符**
  - `*` —— 0 次或多次，贪婪
  - `+` —— 1 次或多次，贪婪
  - `?` —— 0 次或 1 次，懒惰

### 构建方式（CMake）

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

会生成：

- **`regex_cli`**: 控制台程序
- **`regex_tests`**: 单元测试可执行文件

### 运行方式

1. **命令行应用**

```bash
./regex_cli
```

第一行输入正则表达式模式，后续每行输入一个待匹配字符串，程序对每一行输出一行 `true` 或 `false`。

示例：

```text
[a-c]+
abc
d
abcc
```

输出：

```text
true
false
true
```

2. **运行测试**

在 `build` 目录下：

```bash
ctest
```

或直接运行：

```bash
./regex_tests
```
